# Сравнение биндов Niri

В данной таблице приведено подробное сравнение горячих клавиш из дефолтной конфигурации Niri и моей текущей настройки.

**Основные изменения:**
1.  Я вернул привычную логику тайлинговых WM, где `Mod+Shift+...` отвечает за перемещение окон, а не `Mod+Ctrl`.
2. Закрытие окна перенесено на `Mod+Shift+C` (как в dwm "C"lose), вместо неудобного `Mod+Q` (Можно случайно закрыть окно, пытаясь нажать `Mod+1`)
3. Терминал на `Mod+Shift+Return`, добавлены бинды для уведомлений.
4.  Шаг изменения громкости и размера окон уменьшен для более точной настройки.

## 1. Запуск приложений и система

| Действие               | Описание                                 | Default Bind                   | Custom Bind              | Комментарий                   |

| **Лаунчер приложений** | Меню запуска программ (Fuzzel)           | `Mod+D`                        | `Mod+D`                  | Без изменений.                |
| **Центр уведомлений**  | Открыть панель уведомлений (SwayNC)      | -                              | `Mod+N`                  | Добавлено в моем конфиге.     |
| **Блокировка экрана**  | Блокирует сессию                         | `Super+Alt+L` (swaylock)       | `Super+Alt+L` (hyprlock) | Заменен локер на hyprlock.    |
| **Скринридер**         | Вкл/выкл Orca                            | `Super+Alt+S`                  | -                        | Убрано за ненадобностью.      |
| **Справка**            | Показать оверлей с хоткеями              | `Mod+Shift+?`                  | `Mod+Shift+?`            |                               |
| **Выход**              | Завершить сессию Niri (с подтверждением) | `Mod+Shift+E` / `Ctrl+Alt+Del` | `Mod+Shift+E`            |                               |

| **Inhibit Shortcuts**  | "Проброс" клавиш в виртуалку/RDP         | `Mod+Escape`                   | `Mod+Escape`             |                               |

## 3. Навигация и Фокус

| Действие                | Описание                            | Default Bind               | Custom Bind        | Комментарий                                                                     |
| :---------------------- | :---------------------------------- | :------------------------- | :----------------- | :------------------------------------------------------------------------------ |
| **Фокус: Влево/Вправо** | Переключение между колонками        | `Mod+Left/Right` (или H/L) | `Mod+Left/Right`   | HJKL закомментированы в моем конфиге, чтобы освободить место для других клавиш. |
| **Фокус: Вверх/Вниз**   | Переключение между окнами в колонке | `Mod+Up/Down` (или K/J)    | `Mod+Up/Down`      | Тоже самое.                                                                     |
| **Фокус: Монитор**      | Переход на соседний монитор         | `Mod+Shift+Left...`        | `Mod+Ctrl+Left...` | Освободил `Shift` для перемещения окон.                                         |
| **Фокус: Workspace**    | Переключение рабочих столов 1-9     | `Mod+1...9`                | `Mod+1...9`        |                                                                                 |
| **Начало/Конец**        | Фокус первой/последней колонки      | `Mod+Home/End`             | `Mod+Home/End`     |                                                                                 |
